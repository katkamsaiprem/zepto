# Zepto Cart

A React shopping cart application with Redux state management.

## Features

- Add and remove items from cart
- Real-time cart total calculation
- Responsive design with Bootstrap 5

## Technologies

- React 19
- Redux Toolkit
- Vite
- Bootstrap 5

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
